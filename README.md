## Description

This is a web-based platform solve the problem for individual or organisation to share the digital content via blogs instead of traditional ways that is inconvenient and time-consuming. 

This is the back-end service of the project, front-end service can be found at [https://github.com/daviddang1216/write-our-life](https://github.com/daviddang1216/write-our-life)

## Tech-stack used in this repo
NestJs, TypeScript, MySQL, AWS S3, GoogleAPI

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
